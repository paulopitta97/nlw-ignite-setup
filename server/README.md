### Rascunhos (Server)

- `npm init -y`
- `npm i fastify`
- `npm i typescript -D`
- `npx tsc --init`
- `npm i tsx -D`
- `npx tsx src/server.ts`
- `npm i prisma -D`
- `npm i @prisma/client`
- `npx prisma init --datasource-provider SQLite`
- `npx prisma migrate dev`
- `npx prisma studio`
- `npm i @fastify/cors`


### Rascunhos (Client)

- `npm i create-vite@latest -D -g`
- `npm create vite@latest`
- `cd web && npm install && npm run dev`
